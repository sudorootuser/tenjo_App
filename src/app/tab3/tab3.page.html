NHW FITNESS

Es una tienda línea que ofrece los implementos necesarios para la implementación de un gimnasio en casa.

PASION DEPORTIVA

Es una página web que permite la administración y reserva de diferentes espacios deportivos y sus implementos

VOLANDO

Es una página web donde el usuario podría acceder a diferentes opciones de hospedaje, hacer comparaciones que requiera y realizar la reservación

FARMADAY

Es una página web que permite realizar inventario y venta de productos farmacéuticos, con su respectiva facturación

OC SKATEBOARDING

Es una tienda en línea de zapatos de skateboarding la cual cuenta con aplicación para dispositivos móviles y página web.
