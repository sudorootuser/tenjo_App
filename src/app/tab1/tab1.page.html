Esta área se encarga de realizar la programación y puesta en marcha de aplicaciones para dispositivos móviles y páginas web, garantizando que se cumplan las especificaciones brindadas por el cliente.



Nuestras Herrameintas

IONIC

Es un entorno de desarrollo que permite crear aplicaciones para los principales dispositivos moviles ( android,IOS ) desde una ùnica base de còdigo

PHP

Es un lenguaje de programación y una poderosa herramienta para crear páginas web dinámicas e interactivas, en su mayoría que necesiten almacenar información en una base de datos

WORDPRESS

Es un sistema de gestión de contenidos enfocado a la creación de cualquier tipo de página web, por ejemplo:

  • Sitios web comerciales
  • Tiendas online
  • Blogs
  • Portafolios
  • Currículos
  • Foros
  • Redes Sociales
  • Sitios de membresía

VISUAL STUDIO CODE

Es un editor de texto totalmente gratuito que ofrece a los usuarios una herramienta de programación avanzada como alternativa al Bloc de Notas
